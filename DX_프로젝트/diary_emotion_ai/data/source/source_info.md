## 사용하는 컬럼

- `감정_대분류`
  - 모델의 정답 라벨로 사용합니다.
- `감정_소분류`
  - 데이터 생성 시 감정 표현을 다양하게 만드는 보조 정보로 사용합니다.
- `상황키워드`
  - 다이어리 문체 생성 시 상황 문맥으로 사용합니다.
- `사람문장1`, `사람문장2`, `사람문장3`
  - 사용자의 감정 발화입니다. 시스템 응답 문장은 학습에 사용하지 않습니다.

## 변환 방식

원본은 대화체이므로 다이어리 앱에 맞게 다음 형식으로 변환합니다.

- 원문 발화 결합
- 오늘 있었던 일처럼 표현
- 감정 상태를 일기 문장으로 자연스럽게 마무리
- 원본 감정 라벨은 유지

예시:

```text
원본: 일은 왜 해도 해도 끝이 없을까? 화가 난다.
변환: 오늘은 일은 왜 해도 해도 끝이 없을까? 화가 난다. 이런 생각이 계속 남아서 마음이 편하지 않았다.
라벨: 분노
```
